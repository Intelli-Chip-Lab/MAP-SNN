# MAP-SNN
MAP-SNN[1] on N-MNIST and SHD.

## Requirement
- Python 3.6
- Pytorch 
- SnnTorch
- SHD dataste [2]
- N-MNIST dataset [3]

[Online]. Available: http://arxiv.org/abs/2204.09893.
[2] Spiking Heidelberg Datasets https://zenkelab.org/resources/spiking-heidelberg-datasets-shd/
